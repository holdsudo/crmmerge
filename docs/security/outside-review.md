# Outside Review Checklist

## Required External Review

- Application security review
- Cloud configuration review
- Authentication and MFA review
- Authorization and object-level access review
- File encryption and storage review
- Backup and restore review
- Logging and breach-readiness review

## Evidence To Provide

- Latest architecture diagram
- Written Information Security Program
- Incident response process
- Backup and restore evidence
- Latest dependency scan and remediation list
- User access roster and role definitions
- Sample audit and security event logs

## Frequency

- Before public launch
- Annually thereafter
- After any material architecture or data-handling change
